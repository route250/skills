# 色彩パターン（用途別・視覚効果別）

この資料は、目的に応じた配色を選ぶための実務基準。

## 1. データ表現の基本パターン
| 用途 | 推奨パレット | 視覚効果 | 注意点 |
|---|---|---|---|
| 量の増減（低→高） | Sequential（連続） | 強弱・密度が直感的 | 明度差を主軸にする |
| 正負・基準差（±） | Diverging（発散） | 中央基準からの逸脱が分かる | 中央値の意味を明記 |
| カテゴリ比較 | Qualitative（定性的） | 分類の識別性が高い | 色数を必要最小限にする |

## 2. シーン目的別の推奨
| シーン目的 | 色戦略 | 期待する効果 |
|---|---|---|
| 導入（全体像） | 低彩度 + 1アクセント | 認知負荷を抑え、焦点を作る |
| 警告・リスク提示 | 暖色アクセント（赤/橙）を限定使用 | 注意喚起を強化 |
| 比較説明 | ニュートラル地 + 対比2色 | 差分の可視化 |
| 手順説明 | 同系色の明度段階 | 連続性・順序感 |
| 締め・結論 | 主ブランド色 + 高コントラスト文字 | 記憶定着 |

## 3. 可読性とアクセシビリティ
- 通常文字のコントラスト比は少なくとも 4.5:1。
- 大きい文字（約24px以上相当）は少なくとも 3:1。
- 色だけで意味を伝えない。形・位置・ラベルを併用する。

## 4. 実装メモ
- 先にグレースケールで階層を作り、最後にアクセント色を足す。
- 同一シーンで意味の異なる赤を複数使わない。
- 色覚多様性に備え、赤緑依存の符号化を避ける。
